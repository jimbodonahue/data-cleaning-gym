# Jim's Data Gym
This is the first step in my journey to starting a data gym. On this particular page, we are practicing imputing missing values. Choose from an existing data set, or upload your own. Columns are randomly selected, and data are randomly deleted. Try to impute, using the Pandas library (advanced methods currently not availabe).

# The Bigger Picture
The Data Gym will build on this concept, giving users the chance to handle outliers, tackle type errors, and generally practice all aspects of data cleaning. As we build, we will also move into model development, where we leverage both statistical tests and domain knowledge to build the best model for our data.

# Why do we need a gym?
Many of us started with tutorials, and our first experiences with diverse data often do not happen until production. I clearly remember fielding questions from an econometrics professor about missing values. I watched his expression powerlessly as my insufficient response dropped my final project grade a point. Otherwise, I would have achieved a perfect score.

I want to build a platform that empowers data science learners by giving them the chance to see various data sets. This will give them the chance to exercise, to build the mental muscle mass needed to tackle real-world data science projects. Please reach out if you have feedback, questions, or want to join me on this journey!
